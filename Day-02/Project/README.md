## Project

A simple program that calculates how much each person should pay when splitting a bill, including tip.

## üìù Task Description

In this project, the program:

* Displays a greeting message
* Asks the user for the total bill amount
* Asks the user for the tip percentage
* Asks how many people will split the bill
* Calculates the total bill including tip
* Calculates and displays the amount each person should pay

The final amount per person is rounded to two decimal places.
